renderMode

var renderMode: Int
Content copied to clipboard

Gets or sets the rendering mode. When renderMode is RENDERMODE_CONTINUOUSLY, the renderer is called repeatedly to re-render the scene. When renderMode is RENDERMODE_WHEN_DIRTY, the renderer only renders when the surface is created, or when requestRender is called. Defaults to RENDERMODE_CONTINUOUSLY.

Using RENDERMODE_WHEN_DIRTY can improve battery life and overall system performance by allowing the GPU and CPU to become idle when the view does not need to be updated.

Setter can only be called after setRenderer Getter can be called from any thread.
